Class: Automata

Automata.Automata(element, options)

Automata.Automata represents the automata view.
It takes an HTMLElement that will be used to render.

Constructor

new Automata(element, options)

Automata.Automata constructor
Parameters:
Name Type Description
element HTMLElement The element where the automata will be rendered
options Object The options
Properties
Name Type Description
started Boolean True to start the automata
delay Number The delay (in ms) between each automata repaint
height Number The number of vertical cells
width Number The number of horizontal cells
step Number The cell size (in px)
Source:
  • automata.js, line 192
Example
const myAutomata = new Automata.Automata(document.getElementById('automata'), {
     started: false,
     delay: 100,
     height: 100,
     width: 100,
     step: 10
});

myAutomata.setOption('started', true);
